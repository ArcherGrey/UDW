# 前端常用工具

- IDE（开发环境）
  - 代码
    - `vscode`
    - `sublime`
    - `atom`
  - 切图
    - [pxcook](https://www.fancynode.com.cn/pxcook)
    - photoshop
    - pageruler chrome 插件 可以得到页面元素尺寸
  - 接口调试
    - postman
- `npm`
  - anywhere 随时在本地搭建 http 环境
  - CommitLint 检查项目中 Commit message 是否规范
    - 依赖 husky githook 工具
  - nrm npm 镜像管理工具
  - rimraf 快速删除文件
