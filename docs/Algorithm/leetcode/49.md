# 49. 字母异位词分组

## 题目

给定一个字符串数组，将字母异位词组合在一起。字母异位词指字母相同，但排列不同的字符串。

```auto
示例:

输入: ["eat", "tea", "tan", "ate", "nat", "bat"]
输出:
[
  ["ate","eat","tea"],
  ["nat","tan"],
  ["bat"]
]
```

## 题解

### 哈希表

```js
/**
 * @param {string[]} strs
 * @return {string[][]}
 */
var groupAnagrams = function(strs) {
  let m = new Map();
  for (let s of strs) {
    let ss = s
      .split("")
      .sort((a, b) => {
        return a.localeCompare(b);
      })
      .join("");
    console.log(ss);
    if (!m.has(ss)) {
      m.set(ss, [s]);
    } else {
      let sss = m.get(ss);
      sss.push(s);
      m.set(ss, sss);
    }
  }
  return [...m.values()];
};
```
