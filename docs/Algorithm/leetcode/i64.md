# 面试题 64. 求 1+2+…+n

## 题目

求 `1+2+...+n` ，要求不能使用 `乘除法、for、while、if、else、switch、case` 等关键字及条件判断语句 `（A?B:C）`。

```
示例 1：

输入: n = 3
输出: 6

示例 2：

输入: n = 9
输出: 45

限制：

1 <= n <= 10000
```

## 题解

### 递归 + 逻辑运算符

用 `&` 来判断

```JavaScript
/**
 * @param {number} n
 * @return {number}
 */
var sumNums = function (n) {
  n && (n += sumNums(n - 1));
  return n;
};

```

时间复杂度： `O(n)`

空间复杂度： `O(n)`
