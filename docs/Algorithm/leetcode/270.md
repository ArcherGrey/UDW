# 270. 最接近的二叉搜索树值

## 题目

给定一个不为空的二叉搜索树和一个目标值 `target`，请在该二叉搜索树中找到最接近目标值 `target` 的数值。

```auto
注意：

给定的目标值 target 是一个浮点数
题目保证在该二叉搜索树中只会存在一个最接近目标值的数
示例：

输入: root = [4,2,5,1,3]，目标值 target = 3.714286

    4
   / \
  2   5
 / \
1   3

输出: 4
```

## 题解

### 中序遍历 + 数组查询

中序遍历构建有序数组，然后查询数组

```JavaScript
/**
 * Definition for a binary tree node.
 * function TreeNode(val) {
 *     this.val = val;
 *     this.left = this.right = null;
 * }
 */
/**
 * @param {TreeNode} root
 * @param {number} target
 * @return {number}
 */
var closestValue = function (root, target) {
  let nums = [],
    min = Infinity,
    ans = 0;
  function inOrder(root) {
    if (!root) return;
    inOrder(root.left);
    nums.push(root.val);
    inOrder(root.right);
  }
  inOrder(root);
  for (let i = 0; i < nums.length; ++i) {
    if (min > Math.abs(target - nums[i])) {
      ans = nums[i];
      min = Math.abs(target - nums[i]);
    }
  }
  return ans;
};

```

### 二分查找

利用二叉搜索树的特点

```JavaScript
/**
 * Definition for a binary tree node.
 * function TreeNode(val) {
 *     this.val = val;
 *     this.left = this.right = null;
 * }
 */
/**
 * @param {TreeNode} root
 * @param {number} target
 * @return {number}
 */
var closestValue = function (root, target) {
  if (!root) return 0;
  let ans = [root.val, Math.abs(target - root.val)];
  while (root) {
    let t = Math.abs(root.val - target);
    if (t < ans[1]) {
      ans[1] = t;
      ans[0] = root.val;
    }

    // 目标比当前节点大，选择右子树
    if (target > root.val) {
      root = root.right;
    }
    // 小就选择左子树
    else if (target < root.val) {
      root = root.left;
    }
    // 相等直接返回结果
    else {
      return root.val;
    }
  }
  return ans[0];
};

```
