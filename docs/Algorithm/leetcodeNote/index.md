# 递归

- 递归边界
- 子问题和原始问题是相同类型问题
- 子问题会不断缩小范围
- 找出子问题和原始问题的等价关系
- 存在重复计算，可以保存结果进行优化（记忆化）
- 二叉树的递归通常是 `dfs` （`bfs`依赖队列。依赖栈的可以使用递归）
